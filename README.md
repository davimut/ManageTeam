# Team Management Platform

> Una piattaforma web completa per la gestione dei membri del team, progettata con un pannello di amministrazione avanzato per il controllo totale delle operazioni CRUD e una vista utente dedicata organizzata per squadre.

---

## 🚀 Funzionalità Principali

### 🛠️ Pannello Admin (Gestione CRUD)
* **Accesso protetto e gestione risorse:** Interfaccia centralizzata per amministrare l'intero organico.
* **Operazioni CRUD complete:** 
  * **Create:** Aggiunta di nuovi membri inserendo dati anagrafici, ruoli e assegnazione alle squadre.
  * **Read:** Tabelle dettagliate con filtri e ricerca rapida.
  * **Update:** Modifica in tempo reale delle informazioni e dei ruoli dei dipendenti/membri.
  * **Delete:** Rimozione sicura dei membri non più attivi.

### 👥 Vista Utente (Team & Squadre)
* **Visualizzazione per Dipartimenti:** I membri sono raggruppati in modo visivo e intuitivo in base al team di appartenenza.
* **Dettagli Membro:** Schede informative dedicate per ogni utente con competenze, ruolo e contatti.
* **Interfaccia Responsive:** Ottimizzata per la visualizzazione sia da desktop che da dispositivi mobili.

---

## 💻 Stack Tecnologico

* **Backend:** Java, Spring Boot, Spring Data JPA, REST APIs
* **Frontend:** HTML5, CSS3,Thymeleaf
* **Database:** MySQL

---

## ⚙️ Prerequisiti e Installazione

Assicurati di avere installato sul tuo sistema:
* Java Development Kit (JDK) 17 o superiore
* Maven
* MySQL Server
